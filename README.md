## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

Or for development with auto-reload:

```bash
npm run dev
```

The server will start on [http://localhost:3040](http://localhost:3040).
